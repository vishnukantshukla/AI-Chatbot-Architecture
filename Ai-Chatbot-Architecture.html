1
User Authentication
User signs up or logs in through the React frontend. Credentials are validated and JWT tokens are generated for secure session management.
2
State Management & Type Safety
Redux manages global application state with TypeScript ensuring type safety. JWT tokens are stored in httpOnly cookies with proper type definitions for secure session management.
3
Database Operations
User data is stored and retrieved from MongoDB. This includes user profiles, authentication details, and chat history for personalization.
4
Protected Routes
Chat functionality is protected by middleware that validates JWT tokens. Only authenticated users can access AI chat features.
5
AI Processing
User prompts are forwarded to the selected AI model (OpenAI, Gemini, etc.). The AI processes the request and generates contextual responses.
6
Response Delivery
AI responses are formatted and sent back through the backend API to the frontend, where they're rendered in the chat interface.
